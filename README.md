# EKEEL Annotator

The EKEEL Video Annotation System is a comprehensive tool designed for manual and semi-automatic annotation of educational videos. It allows users to add, edit, and manage annotations that can be used for various research and educational purposes. 

It integrates with various NLP and machine learning models to enhance the annotation process, ensuring high accuracy and efficiency.

## Installation

Follow [this guide](https://ekeel-project.github.io/docs/platforms/quickstart/) from the documentation site of the project.
